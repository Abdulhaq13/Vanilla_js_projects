# Vanilla JavaScript Projects

This repository contains a collection of small projects built using **HTML, CSS, and Vanilla JavaScript**.  
The goal of these projects is to practice core JavaScript concepts, DOM manipulation, and browser APIs without using frameworks or libraries.

## Projects

- Color Palette Generator
- Kanban Board (Drag and Drop)
- More projects coming soon...

Each project is placed in its own folder with its own HTML, CSS, and JavaScript files.

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
